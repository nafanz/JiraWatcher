# JiraWatcher

Скрипт позволяющий следить за статусами задач вошедшими в версию.

* [PyCharm 2021.2 Community Edition](https://www.jetbrains.com/ru-ru/pycharm)
* [Python 3.10](https://docs.python.org/3/whatsnew/3.10.html)
  * [Atlassian Python API](https://pypi.org/project/atlassian-python-api)
  * [PrettyTable](https://pypi.org/project/prettytable)
  * [Requests](https://pypi.org/project/requests)
